## 5. 문제 목록

### High

- 일반 배포 전 검사 명령인 `npm run build`가 안정적으로 통과하지 않습니다.
- 어드민 화면은 코드 기준 접근 정책만 확인됐고, 로컬 서버 연결 문제로 실제 클릭 QA가 보류되어 있습니다.
- 실제 토론 기록이 있는 상태의 상태 변경, 삭제, 상세 보기, 사용자별 사용량 집계, 본문 미노출 유지는 아직 검수 전입니다.

### Medium

- 2026-05-12 기준 어드민과 이미지/화면 관련 변경이 많아 모바일 재검수가 필요합니다.
- 기존 기록에는 실제 모델별 토큰 사용량이 없어 비용 화면은 추정값 중심입니다.

### Low

- `front-work-log.md`, `design-work-log.md`는 현재 폴더에 없습니다.
- ESLint 설정은 아직 정식 검증 루틴으로 쓰기 어렵습니다.
