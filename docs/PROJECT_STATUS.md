## 5. 문제 목록

### High

- QA의 실제 브라우저 클릭 검수 기록이 아직 없습니다.
- 새 토론 로직은 아직 최종 구현된 상태가 아닙니다.
- 확정된 결론 구조가 로직 출력과 UI 화면에 아직 최종 반영됐는지 확인이 필요합니다.
- UIUX C가 이미지 기준 화면 설계 문서를 만들고 있고, 이후 로직/프론트 채팅방에서 구현할 예정입니다.
- 실제 토론 기록이 있는 상태의 상태 변경, 삭제, 상세 보기, 사용자별 사용량 집계, 본문 미노출 유지는 아직 검수 전입니다.
- UIUX B 보정 후 실제 긴 답변과 긴 URL이 포함된 토론 기록으로 모바일 결과 화면 QA가 필요합니다.
- 2026-05-13 오전 6시 45분 이후 QA 신규 로그가 아직 없습니다.

### Medium

- production 실제 로그인 클릭은 운영 계정 중복 로그인 잠금 영향을 피하기 위해 아직 진행하지 않았습니다.
- 기존 기록에는 실제 모델별 토큰 사용량이 없어 비용 화면은 추정값 중심입니다.

### Low

- UIUX B가 건드리지 않은 이전 토론 기록 서랍, PWA 설치 안내 카드, 카카오톡 공유 이미지 캐시, 최종 로봇 이미지 에셋은 별도 확인이 필요합니다.
- ESLint 설정은 아직 정식 검증 루틴으로 쓰기 어렵습니다.
