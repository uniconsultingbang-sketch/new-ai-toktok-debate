# AI Talk Talk 로직/프론트 채팅방 전달 문서

## 이 채팅방의 역할

이 채팅방은 AI Talk Talk 프로젝트의 프론트 전체 개발과 핵심 로직 검증 전용 채팅방입니다.

## 먼저 읽을 문서

- `AGENTS.md`
- `PROJECT_HANDOFF.md`
- `docs/handoff/00-next-chat-handoff.md`
- `docs/handoff/AI_3Mind_Council_Dev_Logic_KR.md`
- `docs/handoff/05-prompt-system.md`
- `docs/work-log/front-work-log.md`가 있으면 함께 확인

## 작업 범위

- 토론 시작 흐름
- 심플/심층 모드
- AI 토론 진행 화면
- 결과 보기
- 최근 토론 클릭
- 저장/조회 흐름
- API와 화면 연결
- 프론트에서 발생하는 오류 수정

## 주의할 점

- 디자인 채팅방이 담당하는 세부 UI 스타일 변경은 최소화합니다.
- 어드민 전용 기능은 어드민 채팅방에 넘깁니다.
- QA에서 나온 Critical/High 문제는 우선 수정합니다.

## 작업 후 반드시 남길 기록

`docs/work-log/front-work-log.md`에 아래 내용을 정리합니다.

- 수정한 기능
- 수정한 파일
- 테스트 결과
- 남은 프론트/로직 이슈
