# AI Talk Talk QA 채팅방 전달 문서

## 이 채팅방의 역할

이 채팅방은 AI Talk Talk 프로젝트의 오류 테스트와 검수 전담 채팅방입니다.

## 먼저 읽을 문서

- `AGENTS.md`
- `PROJECT_HANDOFF.md`
- `docs/handoff/07-qa-report-2026-05-10.md`
- `docs/work-log/front-work-log.md`가 있으면 확인
- `docs/work-log/design-work-log.md`가 있으면 확인
- `docs/work-log/admin-work-log.md`가 있으면 확인

## 확인 범위

- 토론 시작
- 심플/심층 모드
- 최근 토론 클릭
- 결과 보기
- 저장/조회
- 모바일 화면 깨짐
- Vercel 배포 전 오류
- 어드민 화면이 있으면 어드민 기본 흐름

## 오류 발견 시 처리 방식

오류를 발견하면 먼저 문제 목록을 만듭니다.

우선순위:

- Critical: 실행 불가, 저장 불가, 주요 화면 접근 불가
- High: 핵심 기능 오작동, 최근 토론/결과 보기 실패
- Medium: UX 불편, 말투 어색함, 화면 일부 깨짐
- Low: 문구, 간격, 세부 스타일

Critical/High는 직접 수정 가능한 경우 수정합니다.
담당 채팅방에 넘기는 것이 안전한 경우 아래 형식으로 전달 문구를 만듭니다.

- 디자인 채팅방 전달용
- 프론트/로직 채팅방 전달용
- 어드민 채팅방 전달용

## 작업 후 반드시 남길 기록

`docs/work-log/qa-work-log.md`에 아래 내용을 정리합니다.

- 테스트 항목
- 통과/실패
- 발견한 문제
- 수정 여부
- 담당 채팅방에 넘길 요청문
- 남은 이슈
