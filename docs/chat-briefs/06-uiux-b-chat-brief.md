# AI Talk Talk UIUX 개발 B 채팅방 전달 문서

## 이 채팅방의 역할

AI Talk Talk UIUX 개발 B 채팅방은 채팅/토론 진행 화면, 결과 화면, 말풍선, 토론방 모바일 사용성만 담당합니다.

- `AGENTS.md`
- `PROJECT_HANDOFF.md`
- `docs/chat-briefs/00-how-to-use-chat-briefs.md`
- `docs/handoff/06-design-system.md`
- `docs/ai-decision-council-design.md`
- `docs/work-log/design-work-log.md`가 있으면 함께 확인

## 담당 범위

- 토론 진행 화면
- AI별 말풍선
- 긴 답변 말풍선 가독성
- 결과 화면
- 최종 결론 카드
- 긴 문장과 링크 줄바꿈
- 토론방 모바일 화면 사용성

## 담당하지 않는 범위

- 로그인 화면
- 홈 화면
- 이전 토론 기록 서랍
- PWA 설치 안내 카드
- API 로직
- AI 토론 생성 로직
- 저장/조회 로직
- 로그인/인증 로직
- 어드민 화면

위 항목은 각각 UIUX 개발 A, 로직/프론트, 어드민 채팅방 담당입니다.

## 작업 기준

- 일반 웹 UI 구현이 아니라 `Image-to-UI Reconstruction Task` 기준으로 작업합니다.
- 방실장님이 준 이미지의 감정선, 부드러운 그림자, 입체감, 조명감, 여백, 모바일 앱 완성도를 보존합니다.
- 기능을 바꾸기보다 토론방을 실제로 읽고 쓰기 편하게 만드는 데 집중합니다.
- 긴 답변이 들어와도 말풍선, 카드, 버튼, 제목이 깨지거나 겹치지 않아야 합니다.
- 모바일 360px, 390x844, 412px 폭에서 확인합니다.

## QA에서 우선 확인할 항목

- 토론방 긴 답변 말풍선 가독성
- AI별 말풍선 구분이 자연스러운지
- 결과 화면 긴 문장 줄바꿈
- 근거 링크나 긴 텍스트가 화면 밖으로 밀리지 않는지
- 최종 결론 카드가 모바일에서 과하게 크거나 잘리지 않는지
- 토론 진행 중 화면이 답답하거나 깨져 보이지 않는지
