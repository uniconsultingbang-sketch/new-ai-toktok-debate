# AI Talk Talk 어드민 채팅방 전달 문서

## 이 채팅방의 역할

이 채팅방은 AI Talk Talk 프로젝트의 관리자 화면 개발 전용 채팅방입니다.

## 먼저 읽을 문서

- `AGENTS.md`
- `PROJECT_HANDOFF.md`
- `docs/handoff/01-service-structure-prd.md`
- `docs/handoff/03-ia-menu-flow.md`
- `docs/handoff/04-api-structure.md`
- `docs/work-log/admin-work-log.md`가 있으면 함께 확인

## 작업 범위

- 관리자 페이지
- 토론 목록 관리
- 사용자 질문/결과 확인
- 저장 데이터 확인
- 운영자가 볼 통계 또는 상태
- 관리자 전용 접근 흐름

## 건드리지 말 것

- 사용자 홈 화면 디자인
- 사용자 토론 화면 디자인
- AI 캐릭터 말투
- 프론트 핵심 UX 방향

## 작업 후 반드시 남길 기록

`docs/work-log/admin-work-log.md`에 아래 내용을 정리합니다.

- 구현한 관리자 기능
- 수정한 파일
- 테스트 결과
- 남은 어드민 이슈
