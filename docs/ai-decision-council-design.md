# AI Talk Talk 디자인 참고 문서

## 1. 제품 톤

AI Talk Talk은 세 AI가 전문가 회의처럼 논리적으로 토론하고, 마지막에 결론을 정리해주는 모바일 우선 서비스입니다. 이 문서는 초기 디자인 참고용이며, 현재 서비스 톤은 가벼운 수다보다 전문가 회의형 토론을 우선합니다.

피해야 할 느낌:

- 딱딱한 B2B 대시보드
- PPT 보고서 화면
- 사이버펑크, 네온, 과한 미래형 그래픽
- 회색 위주의 기업용 UI
- 너무 유치한 예능 자막 스타일

핵심 키워드:

- 크림 배경
- 파스텔 포인트
- 둥근 카드
- 부드러운 그림자
- 작은 3D 캐릭터
- 카카오톡식 말풍선
- 모바일 우선

## 2. 공통 화면 기준

- 기준 폭: 390px-430px
- 최대 폭: 430px
- 배경: `#FFFDF8`, `#FFFAF0`
- 주요 텍스트: `#20233E`
- 서브 텍스트: `#777C91`
- 보라 포인트: `#7B4DFF`
- 카드 라운드: 24px-28px
- 버튼 라운드: 18px-999px
- 폰트: Pretendard, Apple SD Gothic Neo, Noto Sans KR, sans-serif
- 자간: `-0.02em`에서 `-0.04em`
- 말풍선 본문: 15px, line-height 1.52

## 3. 홈 화면

구조:

1. Hero
   - 좌측: `AI Talk Talk`, `3관점 논리 토론`, 설명문
   - 우측: Claude/GPT/Gemini가 테이블에 앉은 귀여운 AI 회의 일러스트
2. 입력 카드
   - 심플/심층 pill 토글
   - 질문 textarea
   - 0/500 카운터
   - `토론 시작` 버튼
3. 최근 토론 카드
   - 최근 토론 5개
   - 파스텔 아이콘
   - 제목, 생성일
   - `더 보기`
4. 하단 장식
   - 연두색 언덕, 작은 풀/꽃 느낌

홈 컬러:

- 배경: `#FFFDF8` + 우측 상단 노란 glow
- 입력 카드: 흰색 92%, blur, 부드러운 그림자
- 심플 활성: 연노랑 `#FFF4BD`
- 심층 비활성: 연회색 `#F5F5F6`
- 시작 버튼: 딥 네이비 gradient `#272B45` to `#171A2D`

최근 토론 아이콘:

- 인사/조직: 보라 사람 아이콘
- AI 수다/가벼운 질문: 민트 잎 또는 말풍선 아이콘
- 사업/제품: 오렌지 가방 아이콘
- 제약/의료: 블루/민트 의료 느낌
- 일반: 핑크 하트 또는 별

## 4. 토론 상세 화면

구조:

1. 상단 상태 영역
   - 뒤로가기
   - 완료/진행/중단 상태
   - 날짜/시간
   - 빠르게
   - 중단
2. 질문 Hero
   - 큰 질문 제목
   - 우측 작은 노란 캐릭터, 노트북/머그/전구 느낌
3. 단계 바
   - STEP 1 안건 확인
   - STEP 2 AI 토론
   - STEP 3 근거 검증
   - STEP 4 최종 권고
4. Live Debate 카드
   - Claude/GPT/Gemini 말풍선
   - Claude/Gemini는 왼쪽, GPT는 오른쪽
   - GPT 말풍선은 연녹색
   - Claude/Gemini 말풍선은 흰색
   - 속마음은 작고 귀여운 pill로 표시
5. Answer 카드
   - `Simple Answer`, `Concept Answer`, `Final Recommendation`
   - 핵심 결론 박스
   - 4개 요약 카드
   - 근거 링크가 있으면 하단에 별도 표시

말풍선 규칙:

- 최대 너비: 화면의 64%-70%
- 꼬리 느낌: 좌우 모서리 radius 차이로 표현
- 캐릭터는 말풍선보다 작게
- 이름은 캐릭터 아래 중앙 정렬
- 시간은 말풍선 옆에 작게
- 긴 회의록처럼 보이지 않게 여백과 줄바꿈을 둔다

AI 캐릭터 색:

- Claude: 오렌지/코랄, 부드러운 수습형
- GPT: 민트/그린, 차갑게 태클형
- Gemini: 화이트/블루, 건조한 중재형

## 5. 구현 체크리스트

- [ ] 모바일 최대 폭 430px 적용
- [ ] 홈 Hero가 포근하고 귀여운가
- [ ] 입력 카드가 한눈에 보이는가
- [ ] 최근 토론이 1행 중심으로 조밀한가
- [ ] 상세 화면에서 반복 입력 영역이 없는가
- [ ] 상세 화면에서 홈 버튼 대신 뒤로가기만 있는가
- [ ] 말풍선이 카카오톡처럼 좌우 교차되는가
- [ ] 속마음 pill이 말풍선과 자연스럽게 이어지는가
- [ ] 최종 결과 카드가 보고서가 아니라 읽기 쉬운 요약처럼 보이는가
- [ ] 제약/사업/인사 주제의 근거 링크 영역이 깨지지 않는가
