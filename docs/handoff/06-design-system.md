# 디자인 시스템 가이드

## 1. 디자인 방향

AI 톡톡은 카카오톡/카카오뱅크 계열의 귀엽고 부드러운 모바일 앱 느낌을 지향합니다.

핵심 키워드:

- 귀여움
- 말랑함
- 크림톤 배경
- 둥근 카드
- 파스텔 포인트
- 작은 3D 캐릭터
- 카카오톡식 말풍선

피해야 할 것:

- 딱딱한 B2B 대시보드
- 기본 Tailwind 카드 느낌
- 사이버/네온/미래형 UI
- PPT 보고서 느낌

## 2. 모바일 기준

- 기준 폭: 390px~430px
- 최대 폭: 430px
- 전체 컨테이너: `width: 100%; max-width: 430px; margin: 0 auto;`
- 페이지 배경: `#FFFDF8`, `#FFFAF0`

## 3. 홈 화면

구성:

- 히어로 이미지
- 입력 카드
- 최근 토론 카드
- 하단 초록 언덕 장식

히어로는 현재 `public/images/ai-toktok-hero-cropped.png`를 사용합니다.

## 4. 카드 스타일

- 큰 카드 radius: 24~28px
- 버튼 radius: 18px 이상
- 그림자: 넓고 부드럽게
- 회색 테두리보다 흰 카드와 그림자 중심

## 5. 타이포그래피

- 폰트: Pretendard, Apple SD Gothic Neo, Noto Sans KR
- 제목: 800~900
- 본문: 500~700
- 자간: `-0.02em` ~ `-0.04em`
- 말풍선: 15px, line-height 1.52

## 6. 말풍선 규칙

- Claude/Gemini는 왼쪽
- GPT는 오른쪽
- GPT 말풍선은 연녹색
- Claude/Gemini 말풍선은 흰색
- 말풍선 최대 너비는 60~68%
- 속마음은 작은 pill로 표시

## 7. 모션 규칙

- 발언은 한 번에 다 보여주지 않습니다.
- 사람이 읽을 수 있도록 순차 공개합니다.
- `빠르게` 버튼은 대기 시간을 줄이는 용도입니다.
- 최종 리포트는 토론 뒤에 등장합니다.

## 8. 향후 디자인 마감

- 서브 화면도 이미지 복원형 기준으로 맞춥니다.
- AI 캐릭터는 실제 이미지 에셋으로 교체 가능하게 유지합니다.
- 최근 토론 목록은 앱 리스트처럼 조밀하게 유지합니다.
