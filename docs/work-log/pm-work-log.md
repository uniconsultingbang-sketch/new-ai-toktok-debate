- 모든 관점은 사용자가 입력한 안건을 사회자가 해석해서 생성합니다.
- 이 로직은 아직 최종 구현된 상태가 아닙니다.

- `주제별로 3개 관점을 자동 생성하고, 각 AI 카드에 장점/단점/인사이트를 표시하는 토론형 분석 서비스`라고 설명하는 것이 맞습니다.
