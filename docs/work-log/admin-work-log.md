- 기본 관리자 계정 ID는 `demo03`이며, 필요하면 배포 환경변수 `ADMIN_USER_ID`로 바꿀 수 있게 했습니다.
- 관리자 비밀번호는 코드에 직접 저장하지 않고 기존 `APP_LOGIN_USERS` 환경변수에 등록된 값을 사용합니다.

### 남은 이슈
- Windows 로컬 환경에서 여러 Next 미리보기/빌드 프로세스가 동시에 켜져 `.next` 산출물이 충돌했습니다. 오래된 3010/3020/3030/3040/3041/3042 서버를 정리해야 안정적으로 빌드와 미리보기를 할 수 있습니다.

- Vercel에도 `APP_LOGIN_USERS` 안에 `demo03` 계정이 포함되어 있어야 실제 배포 주소에서 어드민 로그인이 가능합니다. 비밀번호 값은 문서에 남기지 않습니다.
