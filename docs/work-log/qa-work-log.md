### 통과한 항목

- `/login` 정상 응답을 확인했습니다.
- `demo01` 실제 브라우저 로그인 통과.
- `demo01` 로그인 후 홈 화면 진입 통과.
- `demo01`에서 토론 시작 버튼 클릭 후 `/decisions/:id` 결과 화면 이동 통과.
- 토론이 최종 결론까지 생성되고 `최종 결론`, `이유`, `주의할 점`, `핵심 쟁점`, `현실적인 실행 방향`, `근거 자료`가 표시되는 것을 확인했습니다.
- 홈으로 돌아간 뒤 이전 토론 기록에서 방금 만든 토론이 `완료` 상태로 보이는 것을 확인했습니다.
- 최근 토론 클릭 시 해당 결과 화면으로 다시 이동하고 최종 결론이 유지되는 것을 확인했습니다.
- `demo02` 실제 브라우저 로그인 통과.
- `demo02` 이전 토론 기록에서 `demo01`이 만든 토론이 보이지 않고, 빈 기록 안내가 표시되는 것을 확인했습니다.
- `demo03` 실제 브라우저 로그인 통과.
- `demo03` 로그인 후 `/admin` 접근 통과.
- 어드민 5개 메뉴 클릭 통과:
  - 전체 대시보드
  - 사용자별 사용량
  - 토큰/비용 분석
  - 토론 기록 관리
  - 요금 계산 설정
- 실제 기록이 있는 상태에서 어드민 기록 목록, 기록 상세 보기, 추정 토큰/비용 표시를 확인했습니다.

## 2026-05-13 - 임의 질문 하드코딩 위험 QA

### 작업한 채팅방 역할

- QA 테스트 채팅방

### 테스트 목적

- 시연 중 사장님이 어떤 질문을 넣어도 특정 예시 질문에 맞춘 하드코딩 답변이 튀어나오지 않는지 확인했습니다.
- 로컬 `.env.local`에는 AI API 키가 없어, 이번 테스트는 실제 외부 AI 응답이 아니라 내부 fallback 응답 기준으로 확인했습니다.

### 테스트 질문

- 우리 회사가 유튜브 숏폼 광고를 시작해도 될까?
- 신제품 가격을 기존보다 20퍼센트 올려도 괜찮을까?
- 사무실을 강남에서 성수로 옮기는 게 좋을까?
- 고객 상담에 AI 챗봇을 도입해도 될까?
